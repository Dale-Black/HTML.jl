HTMLStrings.jl

Introduction

HTMLStrings.jl is a Julia package designed to make it easy to create and manipulate HTML strings programmatically.

Installation

using Pkg
Pkg.add("HTMLStrings")

Usage

Here's a simple example to get you started:

using HTMLStrings

html = divv("Hello, world!")
println(html)
